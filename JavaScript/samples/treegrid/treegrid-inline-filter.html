TreeGrid / Inline Filter

Inline filtering allows you to quickly search among treegrid data and locate the specific information quickly. In this example you can filter the treegrid data using the inline filter, which appears in first row. Depending on column data type, a different filter is applied for Date, Numeric and String values.

You can choose a filter operation for each column that has filtering enabled. By clicking on filter operation icon a dropdown list will popup from where you can choose a different operation. In case of Dates, clicking on calendar icon allows you to select a Date.
