Menu / Shortcuts

An example of menu with shortcuts. Some of the menu items have a shortcut key attached. Whenever the combination of keys set in the shortcut is pressed, the menuClick event will fire. Shortcuts will call the same function as the menu item is clicked. In this case, an alert box will popup.

You can create custom shortcuts with different combinations of CTRL, SHIFT, ALT and function or regular keys.

In addition, you can customize the shortcut label with CSS.
