ProgressBar / Overview

IntegralUI ProgressBar is a native Web Component that visualize the progression of an operation. You can customize progress indicator using different colors via CSS. In addition, progress label can have different alignments in relation to progress track.

The following properties and events are supported:

  • allowAnimation - Determines whether changes to the progress value are animated or not
  • customStyle - Specifies an object that contains all custom style settings for the component
  • data - Specifies an object that holds data related to the component
  • enabled - Determines whether the component is enabled or disabled
  • labelAlignment - Specifies the position of progress label in relation to progress track
  • name - Uniquely identifies the component
  • showLabel - Determines whether the progress value label is visible or not
  • state - Specifies the component state: disabled, hovered, etc.
  • value - Holds the current progress value

  • valueChanged - Occurs when value property changes
